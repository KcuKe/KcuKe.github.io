Команды

Все команды доступны только администраторам чата

/ban @user [время] [причина]

Забанить пользователя. Время: 1h, 1d, 7d, forever. Без времени — навсегда.

/mute @user [время] [причина]

Замутить пользователя. Время: 15m, 1h, 24h, 7d. Без времени — 1 час.

/unban @user

Разбанить пользователя раньше срока.

/unmute @user

Размутить пользователя раньше срока.

/warn @user [причина]

Выдать предупреждение. 3 варна = автобан.

/unwarn @user

Снять последнее предупреждение.

/warns @user

Посмотреть историю предупреждений пользователя.

/settings

Открыть панель настроек бота для текущего чата.
